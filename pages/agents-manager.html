When to Use Agents - Decision Tree
IF: Task requires multiple rounds of search/grep (open-ended exploration)
→ Use: Explore Agent (saves context, more efficient)

IF: Planning implementation steps for code changes
→ Use: Plan Agent (structured planning mode)

IF: Complex multi-step task (research + code + execution)
→ Use: General-Purpose Agent (full autonomy)

IF: Simple search, specific file/class, or single operation
→ Use: Direct Tools (no agent needed)

IF: Multiple independent tasks
→ Use: Parallel Agents (single message, multiple Task calls)
